What is the GitHub recruiter?

Let's think about GitHub as a social network, a social network where the users are developers. Developers with skills in particular languages, particular expertice you might be looking for !

Here is where the GitHub recruiter comes to life. A personal open source project to search for this data (developers), filter it and use it to give them the chance to be contacted for more opportunities !

Important: This is a personal open source project and it has nothing to do with the GitHub organization.
